# ecommerce-mern
Fully dynamic MERN eCommerce website with admin panel and Razorpay integration
ecommerce-mern is a fully dynamic MERN stack web application designed for scalable eCommerce operations. The platform includes a powerful admin panel and user interface where all core modules are dynamically managed.
Key Features:
- Dynamic Category Management (Add / Edit / Delete)
- Dynamic Product Management with images, pricing, and stock
- Blog Management System
- Gallery Management System
- User Authentication & Authorization
- Admin Dashboard with full control
- Razorpay Payment Gateway Integration
- Responsive UI (Mobile + Desktop)

Tech Stack:
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB (Atlas)
- Payment Integration: Razorpay

Highlights:
- Fully dynamic data handling
- Clean UI/UX design
- API-based architecture
- Secure payment processing

This project is built for learning, portfolio showcase, and real-world application development.
